NumPy Learning Notebook

This repository contains my hands-on practice while learning the NumPy library for numerical computing in Python. The notebooks include explanations, examples, and exercises covering core NumPy concepts, along with some sample data files used for practice.

Topics Covered
Array Basics (numpybasics.ipynb)
Creating arrays
Getting array dimensions
Array shape
Getting and setting data types
Array size and total size calculation
Array Indexing (arrayindexing.ipynb)
Accessing 1D arrays
Accessing 2D arrays
Accessing 3D arrays
Copy vs View (numpycopyandview.ipynb)
Copying arrays
Viewing arrays
Reshaping and flattening arrays
Joining and splitting arrays
Iteration (iteration.ipynb)
Iterating over 1D arrays
Creating 2D Arrays (main.ipynb)
General Practice (basics.ipynb, students.ipynb)
Working with real sample data (student records, weather data)
Repository Structure
File	Description
numpybasics.ipynb	Core array creation, shape, dtype, and size operations
arrayindexing.ipynb	Indexing into 1D, 2D, and 3D arrays
numpycopyandview.ipynb	Copy vs. view semantics, reshaping, flattening, joining, splitting
iteration.ipynb	Iterating through array elements
main.ipynb	Creating and working with 2D arrays
basics.ipynb	General NumPy practice
students.ipynb	Practice using sample student data
